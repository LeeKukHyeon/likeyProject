- 보관료 발생 (예정)
입고 전 / 입고 후
  • - 신청서의 등록/미등록 여부와 관계없이 구매 물품의 센터 도착 여부 또는 신청서의 상세상태 기준으로 구분되며,일반 보관료의 경우 센터 도착일 기준 무료보관일 30일이 경과될 경우 31일 째 $5.00 부과, 이후 추가 1일마다 $1씩 추가됩니다.
  • - 구매 물품 센터 도착 전 신청서 등록 : 센터 도착일 기준으로 보관일수 카운트
  • - 구매 물품 센터 도착 전 신청서 미등록 : 센터 도착일 (미작성 입고/No data) 기준으로 보관일수 카운트
(무게측정 후) 출고보류
  • - 무게측정 후 출고보류 신청 건은 출고신청 전까지 해당 물류센터에서 보관되며 무료보관일수 15일 경과 시 16일째부터 수수료가 부과됩니다.
미작성 입고/No Data
  • - 미작성 입고/No Data 건은 센터 도착일 기준으로 보관일수가 카운트 되며, 무료보관일 수 15일 경과 시 16일째부터 수수료가 부과됩니다.
  • - 미신청 상태로 무료보관일 초과된 대상은 신청번호 생성 시 발생된 NO DATA 보관료가 해당 결제비용에 포함하여 청구됩니다.
